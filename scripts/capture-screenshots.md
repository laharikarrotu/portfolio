# Screenshot Capture Guide

## Option 1: Manual Screenshot Capture (Recommended)

### For Live Websites:

1. **Open your live website** (e.g., https://smart-buy-v2.vercel.app)
2. **Take full-page screenshots** using:
   - **Chrome DevTools**: Press `F12` → `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows) → Type "screenshot" → Select "Capture full size screenshot"
   - **Browser Extensions**: 
     - [Full Page Screen Capture](https://chrome.google.com/webstore/detail/full-page-screen-capture/fdpohaocaeghifibnepbkbifpbpjgkfj)
     - [GoFullPage](https://chrome.google.com/webstore/detail/gofullpage-full-page-scre/fddbokdfbkbdmndokbopojjefjomocfa)
   - **Mac**: `Cmd+Shift+4` then press `Space` for window capture
   - **Windows**: `Win+Shift+S` for Snipping Tool

3. **Save screenshots** with descriptive names:
   - `homepage.png` - Main landing page
   - `ai-assistant.png` - AI feature interface
   - `dashboard.png` - Admin/user dashboard
   - `performance.png` - Performance metrics (Lighthouse scores)

### For GitHub Repositories:

1. **Take screenshots of**:
   - README.md with project description
   - Code structure
   - Key features in the repo
   - Any demo images in the repo
